## 一、📚 争议 1：训练数据合不合法

### 核心问题
模型公司能不能未经允许，**用全网的书 / 图 / 视频训练自己的模型**？

### 当前主要观点
| 立场 | 代表 |
|---|---|
| **合理使用（Fair Use）** | OpenAI 等多数厂商 |
| **侵权** | 纽约时报 vs OpenAI 案 |
| **需许可 + 付费** | 部分艺术家集体诉讼 |

> ⚠️ 全球还没统一答案。**每年都有新判决在出**。

---

## 二、🎨 争议 2：AI 生成物有没有版权

### 美国
- 美国版权局：**纯 AI 生成 = 不享有版权**
- 但**人类有"显著创造性贡献"**（修改、组合、再创作）→ 可主张版权

### 中国
- 北京互联网法院 2023 年首案：**AI 生成画作有版权**（用户对生成有"独创性投入"）
- 中国整体倾向认可"使用 AI 的人"的版权

### 欧盟
- 偏向人类创造，纯 AI 生成不保护

> 💬 **关键观点**：**"你做了多少"决定"你拥有多少"**。

---

## 三、🎭 争议 3：风格 / 人物 / 商标

### 常见红线
| 风险点 | 例子 |
|---|---|
| 模仿在世画家"画风" | "宫崎骏风" 已被多次起诉 |
| 生成真人头像 | 名人 / 政客 / 公众人物 |
| 复刻品牌 LOGO | 商标侵权 |
| 复刻 IP 角色 | 米老鼠、宝可梦 |

✅ **安全做法**：避开**在世名人 / 知名 IP / 注册商标**。

## 四、🌐 各国主流立场速览

| 国家 / 区域 | 立场 |
|---|---|
| 美国 | 拉锯 + 个案判决 |
| 欧盟 | **EU AI Act**：强监管 + 标注义务 |
| 中国 | 标注 + 备案 + 倾向保护使用者 |
| 日本 | 训练数据"较宽松" |
| 英国 | 推行"文本数据挖掘例外" |

---

## 五、🛡 企业使用 AI 的合规底线

```
[ ] 训练数据：来源合法 + 有授权
[ ] 生成内容：明确标注"AI 生成"
[ ] 用户隐私：脱敏 + 最小必要
[ ] 版权：避免明确 IP / 真人
[ ] 留痕：记录每次生成的 prompt + 模型版本
[ ] 内部规范：员工 AI 使用守则
```

---

## 六、🎁 普通人 / 自媒体的安全做法

| 场景 | 做法 |
|---|---|
| AI 生成图配文章 | 选可商用的平台付费版 |
| AI 写公众号 | 人工改写 ≥30% 再发 |
| AI 复刻名人风格 | **不要** |
| 商用海报 | 双源（Midjourney + 自摄）混合 |
| 提交客户 | 注明哪部分 AI 生成 |

> 🎯 **一句话总结**：**"AI 协作 + 你显著二创"是当前最稳的姿势**。
