# 🟠 用 AI 读论文 / 读财报 / 读合同：3 套结构化阅读法

> 你和 AI 都在"读 PDF"，
> 区别是：你看 2 小时只记住第一页，
> AI 看 30 秒交回一份 1 页结构化摘要。
> **关键是你怎么"派任务"。**

## 一、📜 通用铁律

1. **先让 AI 给框架，再深问细节** —— 别一上来就问细节
2. **要求出表格** —— 比段落更易扫
3. **关键数字必须人工二次核对** —— AI 偶尔会"读错小数点"

> ⚠️ **重要提醒**：涉及金额、日期、签字方的关键字段，**AI 仅作辅助**。

---

## 二、📑 模板 1：读论文

```
你是一位资深科研工作者。
请用以下结构总结这篇论文：

1. 一句话核心贡献
2. 解决了什么问题（背景痛点）
3. 用了什么方法（核心 idea）
4. 实验结果（关键数字）
5. 局限和未来方向
6. 一句话告诉一个高中生它在讲什么

最后给我 3 个值得深挖的关键概念。
```

✅ **适合**：综述、摘要、文献综述前置工作

## 三、💰 模板 2：读财报

```
你是一位资深证券分析师。
请基于这份财报输出：

【核心摘要】营收 / 利润 / 增速 一句话
【关键数字表】
| 指标 | 本季 | 同比 | 环比 |
| 营收 | | | |
| 净利 | | | |
| 毛利率 | | | |
| 经营性现金流 | | | |

【三大亮点】
【三大风险】
【管理层 Guidance 关键变化】
【对比上季有什么明显不同】
```

> 💬 **关键观点**：财报里**最值钱的不是数字，是同比 / 环比 / Guidance 变化**。

---

## 四、📃 模板 3：读合同

```
你是一位有 10 年经验的商业合同律师。
请输出：

1. 合同核心交易摘要（3 行）
2. 我方义务清单
3. 对方义务清单
4. 关键金额 / 时间节点表
5. 违约条款一览
6. 5 个最值得警惕的条款（红字标出风险点）
7. 你建议谈判时争取修改的 3 个地方
```

✅ **适合**：劳动合同、采购合同、保密协议、租房合同

⚠️ **法律生效以人工审核为准**。

---

## 五、🎁 让"读"效率翻倍的 3 个小技巧

| 技巧 | 怎么做 |
|---|---|
| 📌 先问 "目录" | "请先给我这份文档的章节目录" |
| 🎯 二段式提问 | 先要摘要，再深问某一节 |
| 🧪 反向出题 | 让 AI "出 5 道题" 反向检验你是否读懂 |

---

## 六、❌ 常见误区

| 误区 | 真相 |
|---|---|
| ❌ "帮我读完这份 PDF" | ✅ 模糊，AI 给你一坨 |
| ❌ 长文档一次问到底 | ✅ 分块 + 分步 |
| ❌ 全信 AI 数字 | ✅ 数字必须看原文核对 |

---

## 七、🎁 一句话总结三套模板

> 🎯 **论文要核心贡献 + 局限**
> 🎯 **财报要变化 + Guidance**
> 🎯 **合同要义务 + 违约 + 红线**
