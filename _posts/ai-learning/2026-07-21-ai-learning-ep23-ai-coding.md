## 一、🏷 一句话定位

| 工具 | 定位 |
|---|---|
| **GitHub Copilot** | IDE 里的"自动补全副驾驶" |
| **Cursor** | 完全 AI 原生的 IDE |
| **Claude Code** | 命令行里的"AI 工程师"，能跑能改能验证 |

---

## 二、📊 六维对比

| 维度 | Copilot | Cursor | Claude Code |
|---|---|---|---|
| 上手难度 | ⭐ | ⭐⭐ | ⭐⭐⭐ |
| 补全速度 | ⚡⚡⚡ | ⚡⚡ | ⚡ |
| 多文件改造 | 弱 | 强 | **极强** |
| Agent 能力 | 中 | 强 | **顶级** |
| 价格 | $10/月 | $20/月 | 按用量 |
| 适合 | 所有开发者 | 全栈/独立开发者 | 重度任务 / 自动化 |

---

## 三、💼 我自己的真实用法

```
🎯 写小函数 / 重复代码 → Copilot 自动补全
🎯 大段重构 / 跨文件改造 → Cursor 的 Composer
🎯 长任务 / 自动跑测试 / 一句话生成完整 PR → Claude Code
```

> 💬 **关键观点**：**三者不是替代，是分层**——补全、片段、Agent。

---

## 四、🧑‍💼 完全不写代码的人能用吗

可以，但是要建立合理预期：

| 你的需求 | 现在能不能 |
|---|---|
| 写一个个人小工具 | ✅ 完全可以 |
| 改改 Excel 自动化脚本 | ✅ 入门级 |
| 做一个生产级网站 | ⚠️ 能跑，但坑多 |
| 公司核心系统 | ❌ 别 |

> ⚠️ **重要提醒**：AI 写的代码**能跑 ≠ 能上线**。生产环境必须有懂代码的人审过。

---

## 五、🛣 上手路径图

```
第 1 周：装 Copilot，让它陪你写日常脚本
第 2 周：装 Cursor，试 Composer 多文件改造
第 3 周：装 Claude Code（或 Copilot CLI），让它跑完整任务
第 4 周：搞一个真实小项目（不超 500 行）
```

---

## 六、🎁 不写代码也能开始的 4 个练习

1. 📊 让 AI 写一个 **"批量重命名"** Python 小脚本
2. 📥 让 AI 写一个 **下载某网站图片** 的脚本
3. 📈 让 AI 写一个 **Excel 自动汇总** 脚本
4. 🌐 让 AI 给你做一个 **个人主页** 静态网站

跑通了，你就**正式入门了 Vibe Coding**。

---

## 七、❌ 常见误区

| 误区 | 真相 |
|---|---|
| ❌ AI 能完全替代程序员 | ✅ 替代的是"重复劳动"，不是"判断力" |
| ❌ 一句话能出完整项目 | ✅ 需要拆步骤、迭代 |
| ❌ 越贵的越好 | ✅ Copilot 已够 80% 场景 |
| ❌ AI 写的代码不用审 | ✅ 必须审，特别是安全 |

---

## 八、🎁 给非程序员的"AI 编程"心法

```
🔑 把 AI 当"实习生"，而不是"专家"
🔑 小步快跑，跑通一步再说下一步
🔑 永远问"这段代码做了什么"
🔑 遇到错误，把错误信息原样喂回去
```
