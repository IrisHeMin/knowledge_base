## 一、🤔 先讲一个反直觉的事实

OpenAI 内部统计：**同一道题，不同 prompt 之间结果质量差距高达 5-10 倍**。
模型升级一代大约只能提升 30-50%。

> 💬 **关键观点**：**优化 prompt 的 ROI，远高于换更贵的模型**。

---

## 二、✍️ 结构 1：CRISPE（写作类）

```
C - Capacity（设定角色）：你是一位资深公众号编辑
R - Role（强调身份）：擅长撰写打工人爱看的科普文
I - Insight（给背景）：我要写一篇关于"番茄工作法"的文
S - Statement（明确任务）：写一篇 1500 字、结构清晰的稿
P - Personality（语气）：聪明朋友式，专业但有趣
E - Experiment（要求迭代）：先给我大纲，我确认后再展开
```

✅ 适合：写公众号、写邮件、写演讲稿、写文案

---

## 三、📊 结构 2：BROKE（分析类）

```
B - Background：背景信息
R - Role：扮演什么角色（如行业分析师）
O - Objectives：要解决什么问题
K - Key Result：想要的结果格式（表格 / 报告）
E - Evolve：允许迭代/追问
```

✅ 适合：写竞品分析、行业洞察、复盘报告

---

## 四、🎯 结构 3：COSTAR（专业输出）

我个人用得最多的一个：

```
C - Context（上下文）：项目背景、利益相关方
O - Objective（目标）：明确要交付什么
S - Style（风格）：正式 / 口语 / 学术
T - Tone（语气）：积极 / 中立 / 严谨
A - Audience（受众）：CEO / 用户 / 学生
R - Response（输出格式）：1 页 PPT / 邮件 / 表格
```

> 🎯 **一句话总结**：**给 AI 足够的上下文 + 明确的输出格式，质量立刻翻三倍**。

---

## 五、📚 结构 4：SAGE（学习类）

```
S - Scenario：你要学什么（如：理解 Transformer）
A - Aspiration：你想达到什么水平（小白 / 入门 / 进阶）
G - Gap：你现在的盲点是什么
E - Examples：要求模型用类比 + 例子讲
```

✅ 适合：把 AI 当老师，从零学一个新领域

---

## 六、🧭 结构 5：3W1H（决策类）

```
What：我现在面临什么问题
Why：根本原因是什么
What if：有哪些备选方案 + 各自利弊
How：你建议我先做哪一步
```

✅ 适合：职业选择、产品决策、个人重大选择

---

## 七、❌ Prompt 5 大常见误区

| 误区 | 真相 |
|---|---|
| ❌ 越长越好 | ✅ 结构清晰比啰嗦更重要 |
| ❌ 一次问到底 | ✅ 分步问、迭代效果更好 |
| ❌ 不给上下文 | ✅ AI 不知道你的项目 |
| ❌ 不指定输出格式 | ✅ 不给格式 = 让 AI 猜 |
| ❌ 用模糊形容词 | ✅ 用"3 段，每段 100 字"代替"详细一点" |

---

## 八、🎁 拿走即用的万能开头模板

```
你是一位【角色】，擅长【领域】。
我现在的情况是：【背景】
我希望你帮我【目标】，
输出格式请用【表格 / 列表 / 报告】，
语气保持【风格】，
如果有不清楚的，先问我 3 个最重要的问题。
```
