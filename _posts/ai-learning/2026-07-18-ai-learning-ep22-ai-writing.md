## 一、🤖 先讲清楚问题：AI 文最大的毛病不是"错"，是"假"

AI 写出来的文章经常一眼能识破，因为：

| AI 文的特征 | 原因 |
|---|---|
| 句式整齐到诡异 | 训练偏好"流畅" |
| 全是空话套话 | 缺少具体细节 |
| 永远"首先 / 其次 / 最后" | 模板化倾向 |
| 没有真实生活气息 | 没有你的素材 |

> 💬 **关键观点**：**AI 写作好坏 = 你给它"原始素材"质量的好坏**。

## 二、📱 模板 1：公众号文章

```
你是一位深耕"打工人成长"赛道的资深公众号编辑。

【素材】
{把你想写的核心观点 / 例子 / 数据贴上来}

【任务】
写一篇 1500-2000 字的公众号文章。

【风格要求】
- 开头一个真实场景或反常识数据钩子
- "我 + 你"的教练式人称
- 每节配小标题（带 emoji）
- 多用短句、表格、加粗
- 避免"首先其次"等模板词
- 结尾留一个互动话题

【交付方式】
先给我标题 3 个备选 + 大纲，我确认后再展开正文。
```

🎯 **关键点**：分两步交付 = **质量翻倍**。

## 三、📊 模板 2：周报

```
你是一位帮我把"流水账"转成"管理者爱看的周报"的助理。

【本周原始流水账】
{贴上你这周做的所有事，乱序也行}

【输出格式】
一、本周关键产出（3 条，每条带"业务价值"）
二、关键数据 / 指标变化
三、卡点与求助
四、下周 Top 3 优先级

要求：
- 不超过 400 字
- 数字优先、名词精准
- 不要废话
- 用 manager 视角，不要 IC 视角
```

> 🎯 **一句话总结**：**周报不是"汇报做了什么"，是"汇报创造了什么价值"**。

## 四、📧 模板 3：邮件

```
你是一位资深商务沟通顾问。

【背景】
{发件人 / 收件人 / 上下文}

【目的】
我想达成 ____

【约束】
- 字数 ≤200 字
- 语气：专业、克制、不失礼
- 一个明确的 Call-to-Action
- 中文 / 英文 / 双语？

请输出 2 版（保守版 + 略 push 版），我自己选。
```

✅ 三种场景特别好用：催回复 / 谈判 / 拒绝

## 五、🛠 4 个"去 AI 味"的技巧

| 技巧 | 做法 |
|---|---|
| 🍃 加生活细节 | "我地铁上看 / 上周开会时" |
| ✂️ 砍掉所有副词 | "非常 / 极其 / 一般而言"全删 |
| 🔢 多用具体数字 | "提升一倍" → "从 12% 提到 27%" |
| 💬 加一句"反方观点" | 让文章看起来更可信 |

---

## 六、❌ 常见误区

| 误区 | 真相 |
|---|---|
| ❌ 一键生成全文 | ✅ 分步生成质量更高 |
| ❌ 不给素材 | ✅ AI 不知道你的故事 |
| ❌ 不指定字数 | ✅ AI 会写到累 |
| ❌ 不审稿直接发 | ✅ 必须改 30% 才像人写的 |
