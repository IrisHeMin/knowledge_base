# 🟠 AI 画图工作流：Midjourney / SD / Nano Banana 怎么选，怎么用

> 今天的 AI 画图，能精准 P 你的产品图、设计封面、出原画。
> **门槛降了，但用得好不好，差距更大了。**

## 一、🏷 三大流派

| 流派 | 代表 | 一句话定位 |
|---|---|---|
| **闭源 SaaS** | Midjourney / DALL·E 3 | 默认审美最高 |
| **开源可控** | Stable Diffusion / Flux | 可调可炼，自由度最大 |
| **多模态精确编辑** | Nano Banana / GPT-Image / Gemini-Image | **精准 P 图、能改局部** |

---

## 二、📐 按场景推荐

| 场景 | 推荐 |
|---|---|
| 公众号配图 | Midjourney / Nano Banana |
| 朋友圈头像 / 风格化人像 | Midjourney |
| 电商产品图 P 图 / 换背景 | **Nano Banana** |
| 海报 / 物料 | Midjourney + Canva |
| 二次元 / 游戏原画 | Stable Diffusion |
| 批量生成、自由训练 | Stable Diffusion |

---

## 三、🧰 万能 Prompt 公式

```
[主体] + [动作/场景] + [风格] + [构图] + [氛围/光影] + [画质参数]
```

### 实例

```
A young Asian woman walking in a Tokyo street at night,
cyberpunk style, neon lights, medium shot, cinematic lighting,
8k, ultra detailed --ar 16:9
```

> 💡 **小技巧**：**先少描述跑一版**，看 AI 的"默认理解"，再针对偏差加描述。**比一上来写 200 字 Prompt 更高效**。

---

## 四、🎯 最被低估的能力：图生图 + 精准编辑

2026 年最大的变化是 **Nano Banana / GPT-Image 系**：

| 能力 | 举例 |
|---|---|
| **保人物特征改场景** | 同一张脸放进 10 种风格 |
| **指哪改哪** | "把红衣服改成蓝色" |
| **多图融合** | "把这两个人合到一张沙发上" |
| **文字精准嵌入** | 海报上的字不再乱码 |

> 💬 **关键观点**：**电商、设计、运营**这一波被改变最深。

---

## 五、❌ 常见误区

| 误区 | 真相 |
|---|---|
| ❌ 描述越长越准 | ✅ 关键词权重比长度重要 |
| ❌ 一次出图就完事 | ✅ 平均 4-8 次迭代才出能用 |
| ❌ Midjourney 永远最好 | ✅ 精准 P 图 Nano Banana 更稳 |
| ❌ AI 出图直接商用 | ✅ 必须确认平台版权条款 |

---

## 六、⚖️ 商用合规提醒

| 平台 | 商用 |
|---|---|
| Midjourney 付费版 | 可商用 |
| Stable Diffusion 自己跑 | 可商用 |
| 含某人 / 某 LOGO 的图 | 仍可能侵权 |

⚠️ **真人面孔 / 知名 IP / 品牌 LOGO** 是高风险区。

---

## 七、🎁 一套"今天就能用"的工作流

```
🎯 公众号封面：Midjourney 出 4 张 → Photoshop 加字
🎯 产品图换背景：Nano Banana 直接 P
🎯 朋友圈头像 9 宫格：Midjourney --tile + 风格统一
🎯 节日海报：Canva 模板 + AI 主图
```
