本文系统整理沪教版二年级上册数学全册知识点，涵盖7个单元，每个知识点附带练习题与详细解答，帮助学生全面掌握本学期内容。

## 第一单元：100以内的加减法（二）

### 一、两位数加两位数（竖式计算）

**（1）不进位加法**

**知识要点：**
- 个位加个位的和**不满10**，直接写在个位下面。
- 十位加十位的和直接写在十位下面。
- 竖式书写：数位对齐，从个位算起。

**示例**：34 + 25

```
    3 4
  + 2 5
  -----
    5 9
```
个位：4 + 5 = 9。十位：3 + 2 = 5。结果：**59**。

**（2）进位加法**

**知识要点：**
- 个位相加**满10或超过10**，向十位进1。
- 要点：个位满十进一，十位别忘加进上来的1。

**示例**：47 + 36

```
    4 7
  + 3 6
  -----
    8 3
```
个位：7 + 6 = 13，写3进1。十位：4 + 3 + 1 = 8。结果：**83**。

### 二、两位数减两位数（竖式计算）

**（1）不退位减法**

**知识要点：**
- 个位上的数够减，直接相减。

**示例**：68 - 35

```
    6 8
  - 3 5
  -----
    3 3
```
个位：8 - 5 = 3。十位：6 - 3 = 3。结果：**33**。

**（2）退位减法**

**知识要点：**
- 个位上的数**不够减**，从十位退1（借1当10）加到个位再减。
- 注意十位退了1后要减1。

**示例**：52 - 28

```
    5 2
  - 2 8
  -----
    2 4
```
个位：2 - 8不够减，向十位借1，变成12 - 8 = 4。十位：4 - 2 = 2（原来5退1变成4）。结果：**24**。

**练习题 1：**

> 用竖式计算：
>
> （1）56 + 37 = ____
>
> （2）83 - 45 = ____
>
> （3）64 + 28 = ____

**答案：**
（1）56 + 37 = **93**（个位6+7=13写3进1，十位5+3+1=9）
（2）83 - 45 = **38**（个位3-5不够借1，13-5=8，十位7-4=3）
（3）64 + 28 = **92**（个位4+8=12写2进1，十位6+2+1=9）

### 三、加减混合运算

**知识要点：**
- **无括号**：按照从**左到右**的顺序依次计算。
  - 例：35 + 24 - 18 → 先算 35 + 24 = 59 → 再算 59 - 18 = **41**
- **有括号**：**先算括号里面**的。
  - 例：72 -（30 + 15）→ 先算 30 + 15 = 45 → 再算 72 - 45 = **27**
- **小括号的作用**：改变运算的顺序。

**练习题 2：**

> 计算：
>
> （1）45 + 28 - 36 = ____
>
> （2）80 -（24 + 37）= ____
>
> （3）56 -（78 - 43）= ____

**答案：**
（1）45 + 28 = 73，73 - 36 = **37**
（2）24 + 37 = 61，80 - 61 = **19**
（3）78 - 43 = 35，56 - 35 = **21**

### 四、"求多/少几"应用题

**知识要点：**
- **求多几**：大数 - 小数 = 多的部分。
  - 题目关键词："多多少""多几个"
- **求少几**：也是大数 - 小数 = 少的部分。
  - 题目关键词："少多少""少几个"
- **求较多/较少的量**：
  - 知道少的量和多多少 → 多的 = 少的 + 多的部分
  - 知道多的量和多多少 → 少的 = 多的 - 多的部分

**练习题 3：**

> （1）小红有45颗糖，小明有28颗糖，小红比小明多几颗？
>
> （2）果园里有苹果树62棵，梨树比苹果树少18棵，梨树有多少棵？
>
> （3）图书馆有故事书38本，科技书比故事书多25本，科技书有多少本？

**答案：**
（1）45 - 28 = **17**（颗）——求多几，用减法
（2）62 - 18 = **44**（棵）——梨树比苹果树少，梨树更少
（3）38 + 25 = **63**（本）——科技书比故事书多，科技书更多

## 第二单元：认识人民币

### 一、元、角、分的认识与换算

**知识要点：**
- 人民币单位：**元、角、分**。
- 换算关系：
  - **1元 = 10角**
  - **1角 = 10分**
  - **1元 = 100分**
- 单位转换技巧：
  - 大单位→小单位：×10（如 3元 = 30角）
  - 小单位→大单位：÷10（如 50角 = 5元）
- 常见面值：1分、2分、5分、1角、2角、5角、1元、5元、10元、20元、50元、100元。

**练习题 1：**

> 填空：
>
> （1）3元5角 = ____ 角
>
> （2）48角 = ____ 元 ____ 角
>
> （3）1元2角5分 = ____ 分
>
> （4）100分 = ____ 元

**答案：**
（1）3元5角 = **35** 角（3×10+5=35）
（2）48角 = **4** 元 **8** 角（48÷10=4余8）
（3）1元2角5分 = **125** 分（1×100+2×10+5=125）
（4）100分 = **1** 元

### 二、购物情景计算

**知识要点：**

**（1）计算总价**
- 买多样物品：各物品价格**相加**。
- 例：笔记本3元5角，铅笔1元2角 → 总价 = 3元5角 + 1元2角 = **4元7角**。

**（2）找零计算**
- 找回的钱 = 付出的钱 - 总价。
- 例：付5元，买4元7角的东西 → 找回 5元 - 4元7角 = **3角**。

**（3）组合支付**
- 用不同面值的钱凑出指定金额。
- 例：凑出1元5角 → 可以用1个1元+1个5角，或3个5角，或15个1角。

**练习题 2：**

> 解决问题：
>
> 文具店价格表：铅笔2角，橡皮5角，尺子1元3角，笔记本2元。
>
> （1）买一把尺子和一块橡皮要多少钱？
>
> （2）带5元钱买一本笔记本，找回多少钱？
>
> （3）3元钱能买哪些组合的文具？（举1-2个例子）

**答案：**
（1）1元3角 + 5角 = **1元8角**
（2）5元 - 2元 = **3元**
（3）示例①：1把尺子 + 1块橡皮 + 1支铅笔 = 1元3角 + 5角 + 2角 = 2元（还剩1元）
示例②：1本笔记本 + 1把尺子 = 2元 + 1元3角 = 3元3角（超了，不行）→ 1本笔记本 + 4支铅笔 = 2元 + 8角 = 2元8角（可以，剩2角）

**练习题 3：**

> 妈妈给小芳10元钱去买水果。苹果4元5角，香蕉3元2角。
>
> （1）两样都买，一共要多少钱？
>
> （2）还能找回多少钱？

**答案：**
（1）4元5角 + 3元2角 = **7元7角**
（2）10元 - 7元7角 = **2元3角**

## 第三单元：表内乘法

### 一、乘法的引入

**知识要点：**
- **乘法的含义**：求**几个相同加数的和**的简便运算。
- 例：3 + 3 + 3 + 3 = 4个3 → 用乘法写成 **3 × 4 = 12** 或 **4 × 3 = 12**。
- **乘法算式各部分名称**：

- **何时用乘法**：看到"几个几""每份相同"就想乘法。

**知识要点：**

- 例："三五十五" → 3 × 5 = 15，5 × 3 = 15。

### 四、"求几个几"和"几倍"的概念

**知识要点：**
- **几个几**：每份一样多，求总数用乘法。
  - 例：每个小朋友发3支铅笔，有5个小朋友，一共需要几支？→ 3 × 5 = **15**（支）
- **几倍**：A是B的几倍 → A ÷ B；反过来，B的几倍就是 B × 几。
  - 例：苹果有4个，梨的数量是苹果的3倍，梨有几个？→ 4 × 3 = **12**（个）

## 第四单元：角的初步认识

### 一、角的构成

**知识要点：**
- **角**由一个**顶点**和两条**边**（直直的射线）组成。
- 角的标准写法：∠（角的符号）。
- 角的大小与**两条边张开的程度**有关，与边的长短无关。
- 边张得越开，角越大；边合得越拢，角越小。

### 二、角的分类

**知识要点：**
- **直角**：像桌角、书角一样方方正正的角，等于90°。
- **锐角**：比直角**小**的角（尖尖的）。
- **钝角**：比直角**大**的角（张得很开）。
- 大小关系：**锐角 < 直角 < 钝角**。

| 角的类型 | 大小 | 生活举例 |
|----------|------|----------|
| 锐角 | < 90° | 张嘴的剪刀、扇子没完全打开 |
| 直角 | = 90° | 桌角、书本角、黑板角 |
| 钝角 | > 90° | 扇子张得很开、钝角三角板 |

### 三、用三角尺判断角

**知识要点：**
- 三角尺上有一个**直角**（标有小正方形的那个角）。
- 判断方法：
  1. 把三角尺的直角**顶点**对准要判断的角的**顶点**。
  2. 三角尺直角的**一条边**和角的**一条边**重合。
  3. 比较：
     - 角的另一条边在三角尺直角**里面** → **锐角**
     - 角的另一条边和三角尺直角的另一条边**重合** → **直角**
     - 角的另一条边在三角尺直角**外面** → **钝角**

### 四、长方形与直角

**知识要点：**
- **长方形**有4个角，每个角都是**直角**。
- **正方形**有4个角，每个角也都是**直角**。
- 判断一个四边形是不是长方形/正方形，可以用三角尺的直角去比每个角。

**练习题 1：**

> 判断下面的说法对不对：
>

>
> （2）所有的直角都一样大。（ ）
>
> （3）钝角比直角大，比锐角大。（ ）

**答案：**
（1）**×**（角的大小只与张开的程度有关，与边的长短无关）
（2）**√**（直角都是90°）
（3）**√**

**练习题 2：**

> 分类——请将下面的角分为锐角、直角、钝角：
>
> ∠1 比三角尺上的直角小
>
> ∠2 和三角尺上的直角一样大
>
> ∠3 比三角尺上的直角大

> ∠4 是一个尖尖的角

**答案：**
- 锐角：**∠1、∠4**
- 直角：**∠2**

**练习题 3：**

> 数角——在下面的图形中，各有几个直角？
>
> （1）一个长方形有 ____ 个直角。
>

（2）正方形有 **4** 个直角。
（3）三角旗一般有 **0** 个直角（普通三角形）或 **1** 个直角（直角三角形，视具体形状而定）。

## 第五单元：观察物体与方向

### 一、从不同角度观察物体

**知识要点：**
- 同一个物体从**正面、侧面、上面**看，看到的形状可能**不一样**。
- 至少要从**三个方向**观察物体才能全面了解物体的形状。
- 正面看到的叫做**正视图**，侧面看到的叫做**侧视图**，上面看到的叫做**俯视图**。

**举例：**
观察一个长方体盒子（如鞋盒）：
- **正面看**：可能是一个长方形
- **侧面看**：可能是一个较小的长方形
- **上面看**：可能是另一个长方形
- 三个方向看到的形状不同

### 二、东南西北基本方位

**知识要点：**
- 四个基本方位：**东、南、西、北**。
- 方位口诀：**上北下南、左西右东**（在地图上）。
- 相对的方位：
  - **北**的对面是**南**
  - **东**的对面是**西**
- **面向北方**：左手是西，右手是东，背后是南。
- **面向南方**：左手是东，右手是西，背后是北。

**练习题 1：**

> （1）地图上，上面是 ____，下面是 ____，左面是 ____，右面是 ____。
>
> （2）面朝北站立，你的右手方向是 ____，左手方向是 ____。
>
> （3）太阳从 ____ 方升起，从 ____ 方落下。

**答案：**
（1）上面是 **北**，下面是 **南**，左面是 **西**，右面是 **东**。
（2）右手方向是 **东**，左手方向是 **西**。
（3）太阳从 **东** 方升起，从 **西** 方落下。

**练习题 2：**

> 判断对错：
>
> （1）一个物体从不同方向看，看到的形状一定不同。（ ）
>
> （2）在地图上，右边表示东方。（ ）

**答案：**
（1）**×**（有时候不同方向看到的形状可能相同，比如正方体从正面和侧面看都是正方形）
（2）**√**

## 第六单元：数学广场（排列组合与规律）

### 一、简单排列组合

**知识要点：**

**（1）数字搭配**
- 用给定的数字卡片组成不同的两位数。
- 注意：0不能做十位（如用0和3只能组成30，不能组成03）。
- 方法：按顺序排列，不重复不遗漏。

**示例**：用 1、2、3 三张卡片，每次取2张组成两位数。
→ 12, 13, 21, 23, 31, 32，一共 **6个**。

**（2）衣物搭配**
- 上衣和裤子的搭配方法。
- **搭配总数 = 上衣件数 × 裤子件数**。

**示例**：2件上衣（红、蓝）和3条裤子（黑、白、灰）。
→ 搭配方法：红黑、红白、红灰、蓝黑、蓝白、蓝灰 → 共 2 × 3 = **6种**。

**练习题 1：**

> （1）用数字卡片 5、7、9 每次取2张组成两位数，能组成几个？分别是什么？
>
> （2）小红有3件上衣和2条裙子，她每天穿一件上衣配一条裙子，有几种不同的穿法？

**答案：**
（1）57, 59, 75, 79, 95, 97 → 共 **6个**
（2）3 × 2 = **6种**

### 二、周期规律

**知识要点：**
- 一组物体按一定规律**重复出现**，找到一个完整的**循环周期**。
- 判断第N个是什么：用N ÷ 周期长度，看余数。
  - 余数为0，对应周期中的最后一个。
  - 余数为几，对应周期中的第几个。

**示例**：○△□○△□○△□……
周期为"○△□"，长度=3。
第10个是什么？10 ÷ 3 = 3 余 1 → 对应周期第1个 → **○**。

**练习题 2：**

> 按规律排列：红、黄、蓝、红、黄、蓝、红、黄、蓝……
>
> （1）第15个是什么颜色？
>
> （2）第22个是什么颜色？
>
> （3）前20个中有几个红色？

**答案：**
周期"红黄蓝"长度=3。
（1）15 ÷ 3 = 5 余 0 → 对应第3个 → **蓝**
（2）22 ÷ 3 = 7 余 1 → 对应第1个 → **红**
（3）每3个一组有1个红色，20 ÷ 3 = 6 余 2，前6组有6个红色，余下2个中第1个是红色，共 6 + 1 = **7个红色**

### 三、简单推理

**知识要点：**
- 根据给出的条件，用**排除法**推理出答案。
- 方法步骤：
  1. 读懂已知条件。
  2. 从确定的信息入手。
  3. 逐步排除，得到结论。

**示例**：
小红、小明、小丽分别拿了语文、数学、英语课本。已知：
- 小红没有拿语文书
- 小明拿的是英语书

推理：
- 小明 → 英语书（已知）
- 小红不拿语文 → 小红拿数学书
- 小丽 → 语文书

**练习题 3：**

> A、B、C三个小朋友分别喜欢画画、唱歌、跳舞。已知：
> - A不喜欢画画
> - B既不喜欢画画也不喜欢跳舞
>
> 请问A、B、C各喜欢什么？

**答案：**
- B不喜欢画画也不喜欢跳舞 → B喜欢**唱歌**
- A不喜欢画画 → A喜欢**跳舞**（唱歌已被B选）
- C → 喜欢**画画**

## 第七单元：分类与统计

### 一、按标准分类

**知识要点：**
- **分类**就是把具有相同特征的事物归为一类。
- 常见分类标准：
  - 按**颜色**分：红色一类，蓝色一类，绿色一类。
  - 按**形状**分：圆形一类，三角形一类，正方形一类。
  - 按**大小**分：大的一类，小的一类。
  - 按**用途**分：学习用品一类，生活用品一类。
- 注意：**同一组物品可以按不同标准分类**，分类标准不同，结果也不同。

**练习题 1：**

> 下面有这些物品：铅笔、苹果、橡皮、香蕉、尺子、西瓜
>
> （1）按用途分成两类，怎么分？
>
> （2）一共有几种水果？几种文具？

**答案：**
（1）
- **文具类**：铅笔、橡皮、尺子
- **水果类**：苹果、香蕉、西瓜

（2）水果有 **3种**，文具有 **3种**。

### 二、正字法统计

**知识要点：**
- **正字法**：用"正"字来统计数量。每个"正"字代表 **5**。
- 统计步骤：
  1. 确定要统计的类别。
  2. 每出现一个就在对应类别画一笔。
  3. "正"字笔画：横、竖、横、竖、横（5笔=5个）。
  4. 最后数"正"字和多余笔画，得出数量。

**示例**：
统计颜色球的数量——红球出现了8次：正 |||  → "正"=5，再3笔=3，共 5+3=**8个**。

### 三、统计表

**知识要点：**
- **统计表**把分类计数的结果整理成表格，清楚明了。
- 统计表一般包含：**类别、数量（用正字计数或直接写数字）、合计**。

| 类别 | 正字统计 | 数量 |
|------|----------|------|
| 红色球 | 正 \|\|\| | 8 |
| 蓝色球 | 正正 | 10 |
| 黄色球 | 正 \| | 6 |
| **合计** | | **24** |

- 从统计表中可以看出：**哪类最多、哪类最少、一共有多少**。

**练习题 2：**

> 二（1）班同学最喜欢的水果统计如下：
>
> | 水果 | 数量 |
> |------|------|
> | 苹果 | 12 |
> | 香蕉 | 8 |
> | 橘子 | 15 |
> | 西瓜 | 5 |
>
> （1）最受欢迎的水果是 ____，最不受欢迎的是 ____。
>
> （2）喜欢苹果的比喜欢香蕉的多 ____ 人。
>
> （3）一共有 ____ 名同学参加了统计。

**答案：**
（1）最受欢迎的是 **橘子**（15人），最不受欢迎的是 **西瓜**（5人）。
（2）12 - 8 = **4**（人）
（3）12 + 8 + 15 + 5 = **40**（名）
